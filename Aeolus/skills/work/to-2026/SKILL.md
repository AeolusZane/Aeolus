# 复制/移动文档到 FineBI前端2026

目标父页面：**【开发文档】FineBI前端2026**，pageId = `1402947188`，Space = `BI`

## 执行步骤

### 1. 解析参数

从 `$ARGUMENTS` 中提取操作类型和页面信息：

**操作类型判断**（默认为"复制"）：
- 参数中包含"移动"或"move" → 操作类型为**移动**
- 参数中包含"复制"或"copy"，或未指定 → 操作类型为**复制**

**页面信息提取**：
- 纯数字 → 视为 pageId，直接使用
- 文字标题 → 先用 `confluence_search_pages` 搜索找到 pageId
- 支持多个页面（逗号分隔），逐个处理

**覆盖检测**：若参数中含"覆盖"，则在执行前先检查目标父页面下是否存在同名页面，存在则先删除。

### 2. 获取源页面信息

用 `confluence_get_page` 读取每个源页面，获取：
- 页面标题（操作时保持原标题不变）
- 所在 Space

### 3. 执行操作

**若为"复制"**，对每个页面调用 `confluence_copy_page`：

```json
{
  "sourcePageId": "<源页面ID>",
  "newTitle": "<原标题>",
  "parentId": "1402947188",
  "targetSpace": "BI",
  "copyAttachments": true
}
```

**若为"移动"**，对每个页面调用 `confluence_move_page`，将页面移动到目标父页面下：

```json
{
  "pageId": "<源页面ID>",
  "targetParentId": "1402947188"
}
```

### 4. 返回结果

每个页面操作完成后输出：
- 操作类型（复制 / 移动）
- 原页面标题 + 原 URL
- 结果页面标题 + 新 URL
- 复制时额外输出附件复制情况

如果有任何页面操作失败，说明失败原因。
