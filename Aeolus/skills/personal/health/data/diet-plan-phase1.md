# 第一阶段饮食计划（外卖版）

> 制定日期：2026-04-08
> 适用场景：成都成华区，不在家做饭，全外卖
> 阶段目标：3个月内体重从77.2kg降至74kg

## 核心原则

血脂高+脂肪肝，核心是**低油、低糖、高蛋白、高纤维**。
每单下单备注：**少油少盐，不要辣，米饭减半**

---

## 每日三餐方案

### 早餐（7:00-8:00）
外卖搜：`粥铺` / `包子铺` / `轻食`

- 燕麦粥/杂粮粥 + 水煮蛋2个
- 或：全麦包子（菜肉/纯素）+ 豆浆（无糖）

❌ 避开：油条、煎饼、肉夹馍、甜豆浆

---

### 午餐（12:00-13:00）
外卖搜：`蒸菜饭` / `快餐盒饭`

- 1荤（清蒸鱼 / 白切鸡 / 水煮虾）+ 2素（炒时蔬 / 凉拌菜）+ 米饭半碗

❌ 避开：回锅肉、红烧肉、麻婆豆腐（油很多）、炒饭

---

### 晚餐（18:00-19:00）
外卖搜：`轻食沙拉` / `日式料理` / `烫菜`

- 轻食沙拉（鸡胸肉/三文鱼款，换油醋汁，不要奶油沙拉酱）
- 日式：刺身定食 / 照烧鸡腿饭（米饭减半）
- 烫菜/冒菜：选豆腐、鱼豆腐、牛肉、蔬菜，少辣少油

---

## 品类优先级

| 优先级 | 品类 | 原因 |
| --- | --- | --- |
| 🟢 首选 | 日料 / 轻食沙拉 | 天然低油 |
| 🟢 首选 | 蒸菜饭 | 油少，成都到处有 |
| 🟡 可选 | 快餐盒饭 | 选清淡菜品 |
| 🟡 可选 | 面条/米线 | 少放油辣子，汤别全喝 |
| 🔴 少吃 | 串串/冒菜/火锅 | 油太多，直接影响血脂 |
| 🔴 少吃 | 炒菜外卖 | 餐厅炒菜油普遍超标 |

---

## 饮品

- ✅ 美式咖啡（无糖）、绿茶、白开水
- ❌ 奶茶、果汁、瓶装甜饮料

---

## 加餐（饿时去便利店）

- 纯牛奶 / 无糖酸奶
- 原味坚果（一小把，约20g）
- 水煮蛋
